Features

一站式评论洞察

从抓取到分析到报告，全流程自动化。不只是数据，更是可执行的产品决策。

智能搜索匹配

输入"ins"也能找到 Instagram。iTunes API 模糊匹配 + Google Play 交叉验证，双平台精准关联。

全球多国覆盖

支持美国、日本、中国、英国、德国等 10+ 国家，自动检测可用性，多国并发抓取。

AI 深度分析

情感分析、痛点提取、功能归因、关键词去重、评分一致性检测，多维度交叉洞察。

质量自评估

Agent 自动评估分析质量，覆盖率不足或一致性异常时自动改进，最多 3 轮迭代。

可视化仪表盘

6 种交互图表 + 评论浏览器 + 多国家 Tab 切换，数据一目了然。

专业报告输出

Markdown + HTML 双格式，P0/P1/P2 量化行动建议，引用真实评论作为论据。

How it works

四步完成分析

从输入到报告，全程自动化，无需任何配置。

1

输入 App 名字

支持模糊搜索，输入缩写或拼音也能精准匹配

2

选择国家和平台

自动检测各国上架状态，勾选目标市场即可

3

AI 自动分析

并发抓取 + 分批分析 + 质量自评估，全程无需干预

4

获取洞察报告

可视化仪表盘 + Markdown/HTML 报告，一键下载
